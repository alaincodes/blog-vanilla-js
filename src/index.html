So I am learning JavaScript.. here is what I learnt so far and how I understand it (From Eloquent JavaScript).

First we will look for the first four basic value types: number, string, boolean and undefined. They are created by typing in their name (true, null) or value (13, "abc"). You can combine and transform values with operators. Like binary operators for arithmetic (+, -, *, /, and %), string concatenation (+), comparison (==, !=, ===, !==, <, >, <=, >=), and logic (&&, ||), as well as several unary operators (- to negate a number, ! to negate logically, and typeof to find a valu's type) and a tenary operator ( ? : ) to pick one of two values based on a thirs value.
